# Encounter Engine

This repo is a snapshot of what an empty but advanced Rails project
shall look like as of 2022.

## What's included

- Rubocop
- Turbo and Hotwire
- Haml
- Headless browser based system tests
- FactoryBot and Faker
- RubyJard and puts_debuggerer
- RSpec and Cucumber
- Tests parallelisation
- Makefile
- SASS/SCSS
- Bootstrap
